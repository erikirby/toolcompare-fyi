Height Review

Height is the new kid in project management that's betting everything on AI. It auto-prioritizes tasks, closes stale issues, merges duplicate tickets, and even writes task descriptions. The vision is a project manager that requires almost no manual upkeep. It's not fully there yet, but the trajectory is interesting for teams tired of process overhead.

PricingFree / $15/mo Pro / custom Enterprise
Rating★★★★ 4.3/5
Best forproduct teams, small teams, engineering teams

Pros & Cons

👍 Pros

  • AI features are genuinely useful — auto-closing stale issues, suggesting priorities, and flagging duplicates saves real time
  • UI is clean and modern; it's a pleasure to use compared to Jira or clubhouse
  • Auto-closed tasks keep your backlog from becoming a graveyard of good intentions
  • The 'why this matters' AI insight on each ticket helps new team members understand context
  • Built-in docs feature covers basic documentation needs without a separate tool

👎 Cons

  • Newer and less proven than incumbents; integration ecosystem is still growing
  • The AI is helpful but occasionally makes wrong decisions that need manual correction
  • No native time tracking, which is a dealbreaker for agencies and billable-hour workflows
  • Limited customization compared to ClickUp or Notion; you get Height's way of doing things
  • Pricing at $15/seat/month is mid-range; not cheap but not outrageous either

Key Features

  • AI-powered issue management with auto-prioritization and deduplication
  • Project views including list, board, Gantt, and calendar
  • Docs for lightweight documentation linked to projects
  • Sprint planning with AI capacity suggestions
  • GitHub and GitLab integration for development workflow sync
  • Slack integration for creating and updating issues from chat

Best Use Cases

  • Product development for teams that want less process overhead
  • Engineering project management with AI keeping the backlog clean
  • Startup project management when you don't want a dedicated project manager
